Introduction to Version 4

 The SRL Development Team is very proud to present to you SRL 4. SRL4 is completely rewritten and is thus NOT COMPATIBLE with SRL 3 Scripts.  The reason for this is the massive reprogramming of incomplete and inconsistent routines and the deleting of double and even tripple copies of equal functions. The good news is: We did it! SRL is clean now, fast, consistent, and ready to explode under your skilled fingers. SRL now meets our standards more then ever.  But the best news is: SRL 4 is maintained and distributed through a Subversion Repository here at SRL-Forums! This has great many advantages.

  • If RS updates, so does SRL once we fix it (which generally doesnt take more then 24 hours at most) No more waiting for the next "official" release, just update the SVN repository and your are up and running.
  • SRL is maintained through a versioning system. Each time we Commit to the repository, SRL's version number is incremented. You can thus tag your Script with a Version Number like this: rev#44.
  • SRL can expand easily. We could commit for instance a  Bitmap database of most used objects, or maybe even add scripts that will continiously be maintained and updated by the Devs!
  • Working with a Repository separates the men from the boys. Installing and using Tortoise SVN is not easy and will thus not be for everybody. We think this is a good thing. Consider it an antileech measure ensuring our future.
  • There is only one SRL! Download SRL ONLY from the repository. Do NOT download SRL from third party vendors. In the evil hands of scammers SRL can potentially be dangereous for your accounts.
How to install SRL
  1. Download and Install the tortoise svn client from Tigris. 
  2. Restart.
  3. Go the Includes folder and make a new folder named SRL.
  4. Rightclick the SRL folder and choose SVN Checkout from this location:  http://www.srl-forums.com:8080/srl-repos/SRL/SRL
  5. You will see all the SRL files pouring one-by-one  to your local SRL folder Done!
New Structure
SRL has been remoddeled into three distinct lvels. TheFirst Level contains lowlevel SCAR routines for generic autoing, like Math, Mouse Movement and Color Clicking routines. The Second Level contains specific RS interface and OCR routines. Level Three has all the High Level functions like MapWalking, AntiRandoms, Banking, Symbol and many many more.

ChangeLog
SRL is fixed and completely compatible with the new RS-Interface. We rewrote text.scar, gametab.scar and inventory.scar for that purpose. Most Globals are removed. The PlayerArray has been modified. Many useless and double/triple fuctions have been removed.



Changes from Globals.scar

Removed All Markers like LogOutMark, RandomChatMark, KillScriptMark etc. Even Mark is gone. Removed ST Script Start Time. Use  DIVI function GetTimeRunning; instead; The User Array is redirected to Players.scar.
Removed dtmPlant, dtmForester, Frog, CWindow, Damien , TheFile, OldLine, TheLine,  TheName moved to AutoRespond.scar (Empty in public version). Removed All AntiBan constants. Removed All GlobalStats Constants.


Changes in Math.scar

NearbyPointInArray by Wizzup? is now a small plugin, for even faster Object Finding.


Changes in  Mouse.scar

MouseFlag.scar is split into Mouse.scar and Flag.scar. All Flag routines in Flag.scar (FlagChat.scar) WindMouse is now the de facto standard in Mouse Movevements.


Changes in Color.scar

Removed the FindMSColor/MMColor and FindMSColortol/MMTol. Too simple for a function.


New Players.scar

New Player Array structure. Removed Player.Fight, Ore and WaitTime. Deleted Player.Boolean1-3, Integer 1-3, Stirng1-3, Ext 1-3. Added Array of Booleans, Integers, Strings and Extended. Added InitPlayerArray, LoadPlayerArray.


Changes in Timing.scar

ScriptTime, Seconds, Minutes and ChangeWait.


Changes in GameTab.scar

GameTabs are now controlled by a TPoint system. Adjusted all GameTab functions. Adjusted SetFightMode, GetCombatLevel, SkillCoords, GetXP,  EquipmentCoords, WaringItem, TakeOff, SetBar and SetRun.


Changes in Text.scar

ChatText is now controlled by a TPoint system. Adjusted all Chat and OCR functions accordingly. Added TextCoords, GetUpTextColor, ChatsBlackText, IsChatBlackTextAnyLine, GetBlackChatMessage, FindBlackChatMessage. Remoddeled ClickOption.
Removed Popup, Option2, IsChatMessage. Added IsUptextMultiCustom. Adjusted ClickNPCChatText. Renamed CLickText2 to ClickText.


Changes in Inventory.scar

Inventory Slots are now controlled by a TBox system. All Inventory functions are adjusted accordingly. Added InvBox. Adjusted MMouseItem, MouseItem, ExistsItem, CountItemBMPtol, CountItemBMPMasktol, ClickAllItemsColorTolWait, ClickAllItemsBMPTolWait, ClickAllItemsBMPMaskTolWait. Removed InventoryCount, UseItem, DropTo, DropInvPosition.


Changes in Object.scar

Added FindObjCustom. Removed FindObjMulti, FindObj3.


Changes in FlagChat.scar

Deleted swap, MouseUntilFlag. Remodded SetChat, Chatsoff.


Changes in Login.scar

Remoddeled Login, LogOut. Deleted NextPlayerOrder, LogoutEvery, HighestAngle. Added FindRS, FindLoginScreen, WelcomeToRunescape.


Changes in MapWalk.scar

Tweaked MakeCompass. Deleted DTM3Flag, added DTMFlag.


Changes in Bank.scar

Added OpenBankFast. Remoddeled Deposit.


Changes in Object.scar

Added FindObjCustom. Removed FindObjMulti, FindObj3.


Changes in RC.scar

Tweaked RCReadCommand, RCFindFriend, RCSetNick.


Changes in AutoColor.scar

Tweaked FindWaterColor,  FindRoadColor, FindVarrockRoadColor,  FindFallyRockColor, FindRockColor, FindStoneColor, FindDirtRoadColor, FindBridgeColor, FindLadderColor.


Changes in AntiRandoms.scar

Tweaked SRLRandomsReport. Deleted ScapeRune, FindName, FindPlant, WeAreDead, FindTrade, BoredHuman, RandomMovement, Wait2. New Pinball-Solver.


Changes in ChatRandoms.scar

Tweaked GetNPCEvent.
Compatibility Issues
x and y. We Removed the Global Variables x and y, since they caused much internal trouble. Just put var x,y:Integer at the top of your script if it complains.
Player Array. We removed the obsolete entries in the Player Record, and replaced them with user assignable arrays. Thus Player.Boolean1 is now replaced with Player.Boolean[0]
ChooseOption. ChooseOption no longer needs x,y, since they where useless. Thus ChooseOption(x,y,'text'); becomes ChooseOption('text');
BenMouse is replaced by WindMouse and RsNMouse is removed. Use MouseSpeed range from 10 - 50.
ClickAllItem series now returns the number of Items clicked.
MouseBox now uses Boolean for Left or Right Clicking.
Global var HowManyPlayers is  internally set.
